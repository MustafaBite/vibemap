# VibeMap Product Overview

VibeMap is a mood-based content recommendation desktop application built with C# Windows Forms. The application analyzes user mood input and provides personalized recommendations for movies, TV series, and games based on emotional state.

## Core Features
- User authentication (login/register)
- Mood analysis and detection
- Smart recommendation engine with local keyword processing
- Content catalog browsing (movies, series, games)
- User preference tracking and history
- SQLite database for user data and content storage

## Target Platform
Windows desktop application targeting .NET Framework 4.7.2 with DevExpress UI components for enhanced user experience.

## Business Logic
The application uses a Smart Recommendation Engine that analyzes user mood input through keyword detection and maps it to appropriate content categories. It tracks user interactions to avoid duplicate recommendations and maintains separate databases for user data and content catalog.
